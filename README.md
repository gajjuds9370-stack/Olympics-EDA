🏅 Olympics Data Analysis & Tableau Dashboard
📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) and building an interactive Tableau dashboard using the Olympics dataset.

The goal of this project is to transform raw Olympic data into meaningful insights through data visualization and storytelling.

I created a complete “Story of Olympics” consisting of three dashboards, each focusing on different aspects of the dataset.

🎯 Objectives

Analyze historical Olympics data

Perform EDA on athletes and medal trends

Build interactive dashboards using Tableau

Present insights in a clear and visual format

📊 Dashboard Structure
🌍 1. Map & Medal Tally Dashboard

Country-wise medal distribution using map visualization

Overall medal tally (Gold, Silver, Bronze)

Helps understand global Olympic performance

📈 2. Overall EDA Dashboard

This dashboard focuses on athlete-level analysis:

👥 Total number of athletes

🌎 Number of participating countries

⚥ Gender distribution (Pie Chart)

📊 Age distribution (Histogram)

📉 Height vs Weight relationship (Scatter Plot)

This provides insights into athlete demographics and physical attributes.

🏆 3. Performance Analysis Dashboard

📅 Year-wise medal tally

🥇 Top performing athletes

🏅 Top sports by medals

This dashboard highlights performance trends and dominance in sports over time.

🛠️ Tools & Technologies Used

Tableau – for data visualization and dashboard creation

Excel / CSV Dataset – for data handling

Data Cleaning – preprocessing before visualization

📂 Dataset

The dataset contains historical Olympic data including:

Athlete details (age, gender, height, weight)

Country participation

Medal records

Such datasets typically include 120+ years of Olympic history data from 1896 onwards, covering athletes and results across events

💡 Key Insights

Medal distribution varies significantly across countries

Gender participation has increased over time

Certain sports consistently dominate medal counts

Physical attributes like height and weight vary by sport

🚀 Learning Outcomes

Through this project, I learned:

How to perform EDA using visualization tools

How to build interactive dashboards in Tableau

Importance of data storytelling

Converting raw data into actionable insights


🙌 Conclusion

This project helped me take a strong step into data analytics and visualization by working on a real-world dataset and building an end-to-end dashboard.
